#Telegram Bot with PostgreSQL Integration
Универсальный Telegram бот на базе aiogram 3.x, использующий асинхронную архитектуру, систему Middleware и интеграцию с базой данных PostgreSQL.

##Технологический стек
 * Язык: Python 3.10+
 * Фреймворк: aiogram
 * База данных: PostgreSQL
 * Драйвер БД: asyncpg (через Connection Pool)
 * Окружение: python-dotenv (безопасное хранение секретов)
 * Логирование: Стандартный модуль logging

##Основные возможности
 * Clean Architecture: Использование роутеров (Router) для разделения логики.
 * Middlewares: Реализована передача пула соединений БД в каждый хендлер через внешние и внутренние мидлвари.
 * Безопасность: Конфигурация через переменные окружения (.env).
 * Логирование: Отслеживание состояния бота и ошибок в реальном времени.
   
##Быстрый запуск
 * Клонируйте репозиторий:
   git clone https://github.com/MrOasisIT/Sender-bot-with-admin-menu.git
cd название_проекта

 * Установите зависимости:
   pip install -r requirements.txt

 * Настройте переменные окружения:
   Создайте файл .env в корне проекта и заполните его:
   TOKEN=ваш_токен_бота
DB_URL=postgresql+asyncpg://user:password@host:port/dbname

 * Запустите бота:
   python main.py

##Структура проекта
 * main.py — Точка входа в приложение.
 * config.py — Загрузка и валидация настроек.
 * handlers/ — Модули с обработчиками команд и сообщений.
 * middleware/ — Промежуточные слои для обработки обновлений.
 * database/ — Инициализация пула соединений с БД.
